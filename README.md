# piano-assessment

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 8.3.8.

## Development server

Run `ng serve --open` for a dev server.`

## Run locally

Once you cloned the app, run below commands:

cd piano-assessment

npm install OR yarn install

ng serve --open
